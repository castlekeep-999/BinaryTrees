# BinaryTrees

Binary Search Tree (BST) - DSA Practice

This repository contains my solutions to Binary Search Tree (BST) problems implemented in C++.

The purpose of this repository is to strengthen my understanding of BST concepts, improve problem-solving skills, and prepare for coding interviews and competitive programming.

## Topics Covered

- BST Traversals
  - Inorder Traversal
  - Preorder Traversal
  - Postorder Traversal

- Basic Operations
  - Search in BST
  - Insert into BST
  - Delete Node in BST

- BST Validation
  - Validate Binary Search Tree
  - Recover Binary Search Tree

- Advanced Problems
  - Lowest Common Ancestor (LCA)
  - Kth Smallest Element
  - BST Iterator
  - Convert BST Problems
  - Range Queries
